## 🚀 Featured Projects

### 🤖 Robotic Reliability Intelligence

Machine-learning and analytics workflows for analyzing large-scale robotic telemetry, detecting operational anomalies, identifying spatial hotspots and supporting reliability investigations.

**Technologies:** Python, SQL, Scikit-learn, clustering, anomaly detection, spatial analytics and time-series analysis

---

### 🦾 Multi-Robot Embodied Intelligence

A multi-robot simulation framework for navigation, coverage, coordination and perception-driven decision-making in dynamic environments.

**Technologies:** Python, ROS, Gazebo, PyTorch, RGB/depth processing and multi-agent systems

---

### 🕸️ SITRUS

A simulation and analytical framework for sensor deployment in irregular environments, focused on coverage, connectivity, energy efficiency and spatial optimization.

**Technologies:** Python, Contiki-NG, Cooja, optimization and wireless sensor networks

---

### 🤖 LLM-Based Information Retrieval and Question Answering

A modular agent framework integrating retrieval, reasoning, tool invocation and memory for multi-step question answering and troubleshooting.

**Technologies:** Python, Hugging Face, BM25, transformers, RAG, prompt engineering and AI agents
